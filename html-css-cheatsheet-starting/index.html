HTML Reference

Tables

Tables are containers for multi-demensional data (i.e. 2-d) content arranged in columns and rows.


Table Tags

Tag Name Description
< table > Table Wrapper element for all HTML tables
< thead > Table Header Contains information for the table heading columns and cell content.
< tbody > Table Body Contains all the rows and columns for the body of the table.
< tr > Table Row Contains all table data for a single row of content in the table.
< td > Table Data Contains all data for individual cells in a table row.
< tfoot > Table Footer Contains and identifies the footer of the table

Table Attributes

Attribute Name Function
colspan Column Span Defines how many columns a td element should span across.
rowspan Row Span Defines how many rows a td element should span across.


Elements

Elements are the term used for the combination of opening tag + content + closing tag.


Attributes

Attribute Recommended Usage Function
'class' General Indentification Identifies multiple elements to make it easier to style groups of elements in CSS.
'id' Specific Identification Uniquely indentifies element for more precise identification when styling.
'href' Links Specifies an internal or external link to resources.
'src' Images Specifies the path to an image (used inside the < img > element)
'width' and 'height' Images Specifies the width and height of an image (in pixels)
'alt' Images Specifies alternate text for an image
'style' Inline Styling Used to specify CSS styling inside the html document. (not recommended)
'lang' Language Indentification (inside html tag) Declares language of the webpage
'title' Extra Information Defines additional information about an element (displayed as a tooltip when mousing over the element).
Recommendations: Always use lowercase attribute names; W3C recommends double quotations ("...") around attribute values.


"Flavor" Tags

Favicon

A favicon image is displayed to the left of the page title in the browser tab. To ad a Favicon follow the steps below:

  1. Either save your favicon image to the root directory of your webserver, or create a folder in the root directory called images, and save your favicon image in this folder (A common name for a favicon image is "favicon.ico").

  2. Add a link element to your "index.html" file, after the title element, like this:

    • < title >My Page Title< /title >
      < link rel="icon" type="image/x-icon" href="/images/favicon.ico" >

  3. Save the "index.html" file and reload it in your browser. Your browser tab should now display your favicon image to the left of the page title.


CSS Reference



Basic Selectors

Selectors are used to select elements by their tag name, id attribute, classes, attributes themselves, and by hierarchy.


Selector Description Code Example
Type Selector Selects all elements that match the specified type or element name.

Can selector multiple elements by separating them with a comma ( , ). 		input {
color: green;
}

h1, h2 {
color: green;
}
Class Selector Selects elements based on the value of their 'class' attribute.

Can also be used to select more specific elements by adding the element name then the class value, this will select all elements that contain the corresponding class value. 		.valid {
color: green;
}

a.valid {
color: green;
}
ID Selector Selects all elements with the same id attribute value. #start {
color: green;
}
Universal Selector Selects ALL elements. * {
color: green;
}
Attribute Selector Selects elements based on the value of ONE of their attributes. In this context the following applies:
  • [attribute]: The attribute occurs in the respective element.
  • [attribute=value]: The attribute occurs AND has the exact specified value.
 [type="text"] {
color: green;
}
